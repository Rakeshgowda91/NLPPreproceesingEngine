# NLP Preprocessing Engine (Advanced)

## 📌 Overview
This project implements a robust Natural Language Processing (NLP) preprocessing pipeline using Python. It is designed to clean and normalize raw text data for machine learning and NLP applications.

---

## 🚀 Features

- Text cleaning (lowercase, punctuation, numbers removal)
- URL removal
- Email removal
- Emoji handling
- Extra whitespace normalization
- Tokenization using NLTK
- Stopword removal
- Stemming (Porter Stemmer)
- Lemmatization (WordNet Lemmatizer)
- Modular and reusable functions
- Handles real-world noisy text data

---

## 🛠️ Technologies Used

- Python
- NLTK (Natural Language Toolkit)
- Regex (re module)
- Emoji library
